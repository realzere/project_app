# data/models/

Модели данных — описывают структуру объектов в приложении.

| Файл | Описание |
|------|----------|
| `user_model.dart` | Модель пользователя с `fromJson`, `toJson`, `fromFirebaseUser` |

## Правила для моделей

1. Расширяют `Equatable` — для корректного сравнения в BLoC.
2. Все поля — `final` (иммутабельность).
3. Имеют фабричные конструкторы `fromJson` / `toJson` для сериализации.
4. Имеют статическое значение `empty` вместо `null` (удобнее в BLoC).

# widgets/

Составные виджеты, привязанные к бизнес-логике приложения.

| Файл | Описание |
|------|----------|
| `google_sign_in_button.dart` | Кнопка «Войти через Google» с SVG-логотипом |
| `user_avatar.dart` | Аватар пользователя с fallback-иконкой при отсутствии фото |

## Чем отличается от ui_kit/?

- **ui_kit/** — «чистые» UI-компоненты, не знающие про бизнес-логику.
- **widgets/** — виджеты, которые знают про конкретную задачу (например, кнопка входа через Google включает локализацию и конкретный SVG).

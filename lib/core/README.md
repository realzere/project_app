# core/

Ядро приложения — код, который **не зависит** от конкретных фич (экранов).

Сюда входят:
- **constants/** — глобальные константы (имена, размеры, пути к ассетам).
- **localization/** — поддержка нескольких языков (RU / EN).
- **router/** — навигация между экранами (GoRouter).
- **services/** — обёртки над внешними сервисами (Firebase Auth, Firestore).
- **theme/** — цвета, стили текста, тема приложения.
- **utils/** — утилиты (логгер и т.д.).

> Правило: если код нужен **нескольким** фичам — он живёт в `core/`.
> Если только одной — в папке этой фичи.

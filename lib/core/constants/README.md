# core/constants/

Глобальные константы приложения.

| Файл | Что хранит |
|------|-----------|
| `app_constants.dart` | Имя приложения, тайминги, размеры, индексы навигации |
| `asset_paths.dart` | Пути к SVG, картинкам и другим ассетам |

> **Зачем?** Чтобы не разбрасывать «магические» строки и числа по коду.
> Если нужно поменять размер аватара — меняем **только здесь**.

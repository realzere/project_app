# features/

Фичи (экраны) приложения. Каждая фича — отдельная папка.

```
features/
├── app/       — корневой виджет + глобальный BLoC авторизации
├── splash/    — сплэш-экран (проверка авторизации при запуске)
├── login/     — экран входа (кнопка Google Sign-In)
└── home/      — домашний экран с вкладками (Main + Profile)
```

## Структура фичи

```
feature_name/
├── bloc/      — BLoC (event, state, bloc) — бизнес-логика фичи
└── ui/        — виджеты и экраны
```

## Как добавить новую фичу

1. Создайте папку `features/my_feature/`.
2. Добавьте `ui/` для экранов и `bloc/` для логики (если нужна).
3. Зарегистрируйте маршрут в `core/router/app_router.dart`.
